HTML:
Hyper text markup language is the standard language used to create web pages
Heading Tags:
  • heading tags are used to define the heading and subheadings
  • Heading tags are defined by h1-h6, where h1 is the biggest and h6 is the smallest
Paragraph Tag:
  • paragraph tags are used used to tell the browser how the content should be displayed as title, or subheadings, or as paragraphs.
  • p tag is defined as paragraph in html, it is blockline element
Tables:
  • The "table" tag defines an HTML table.
  • An HTML table consists of one table element and one or more "tr", "th", and "td" elements.
  • The "tr" element defines a table row, the "th" element defines a table header, and the "td" element defines a table cell.
Blockline Elements:
  • A block level element is always start on a new line and the browser automatically add some space
Inline Elements:
  • A inline elements does not start on a new line and it only occupies the required text space
Formatting Tags:
  1. Bold tag makes the text Bold
  2. Italic tag makes the text Italic
  3. strong tag this is a logical tag which tells the browser the text is important
  4. Mark tag used to highlight the text
  5. underline tag is used top underline the text
  6. superscript tag is used to display the content slightly above the normalline
  7. subscript tag is used to display the content below the Normalline
  8. Delete tag is used to display the deleted content
  9. Blockquote tag is used to define a block of text that is a
    EX: this is a direct quotation
Structure of a URL:
  1. URL stands for uniform resource locator is the address of a unique resource on the internet
  2. A URl is composed of different parts, some mandatory and others optional
  3. The first part of the URL is the scheme which indicates the protocol that the browser must use to request the resourceAfter that there is a domain. In domain there are sub domain, second level domain, top level domain
  4. After that there is a domain. In domain there are sub domain, second level domain, top level domain
  5. Follows the path and parameters are given after that
Anchor tag:
  1. The tag defines a hyperlink, which is used to link from one page to another
  2. An unvisited link is underlined and blue
  3. An visited link is underlined and purple
  4. An active link is underlined and red
Image tag:
  1. The Html image tag is used to embed an image in a web page
  2. Images are not technically inserted into a webpage; Images are linked to web pages
  3. The image tag creates a holding space for the referenced image
